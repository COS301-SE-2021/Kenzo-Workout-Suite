Back Pose Studio If you are a first time user, please read the following before proceeding. Instructions 1. First choose a body part from the drop down menu

2. Now, use sliders to control the body part's rotation about the X, Y and Z axes

3. Once a desired pose is created, simply click/press on one of the red frame buttons to save to that frame

4. To clear a frame, simply press the green frame button and confirm the deletion of that frame

5. At least one pose must be framed in order to create a workout
